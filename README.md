<h2 style="font-weight:300">💬 Changes</h2>

-   Using **Sass** (`.scss`) instead normal **CSS** files;
-   Use **Prettier** and add configuration file (`.prettierrc`);
-   In layout I made some changes, like change header background to a dark gray.

<br />
<br />

<h2 style="font-weight:300">How to Run</h2>

-   To run the development server use the command `yarn dev` or `npm run dev`;
-   To build the project to production use the command `yarn build` or `npm run build`.
